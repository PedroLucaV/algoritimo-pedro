Operadores Comparacionais

  • ==: igual. Retorna verdadeiro caso os dados contenham o mesmo conteudo;
  • !=: Diferente. Retorna verdadeiro caso os dados contenham conteudos Diferente;
  • >: Maior que: Retorna verdadeiro caso um seja maior que o outro;
  • <: Menor que: Retorna verdadeiro caso um seja menor que o outro;
  • >=: Maior ou igual que: Retorna verdadeiro caso um seja maior ou igual ao outro;
  • <=: Menor ou igual que: Retorna verdadeiro caso um seja menor ou igual ao outro;
  • ===: Estritamente Igual: Retorna verdadeiro caso seja o valor e tipos iguais;
  • !==: Estritamente Diferente: Retorna verdadeiro caso seja o valor e tipos diferentes;
